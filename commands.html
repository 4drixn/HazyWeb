Lista de Comandos

🎟️ Sistema de Tickets

  • /tickets - Inicia el sistema de tickets.
  • $add @usuario - Agrega a un usuario al ticket.
  • $remove @usuario - Remueve a un usuario del ticket.
  • $addrole @rol - Agrega un rol al ticket.
  • $removerole @rol - Remueve un rol del ticket.
  • $delete - Cierra el ticket y envía el transcript.
  • $rename - Cambia el nombre de un ticket.

🔨 Moderación

  • /ban @usuario [razón] - Banea a un usuario.
  • /unban [id] [razón] - Desbanea a un usuario.
  • /kick @usuario [razón] - Expulsa a un usuario.
  • /mute @usuario [tiempo] [razón] - Mutea a un usuario.
  • /unmute @usuario - Desmutea a un usuario.
  • /purge [cantidad] - Borra mensajes en masa.

🛠 Administración

  • /canal bloquear - Bloquea un canal.
  • /canal desbloquear - Desbloquea un canal.

💡 Utilidades

  • /sugerir [texto] - Envía una sugerencia.

📜 Misc

  • /verificar - Inicia el sistema de verificación.
  • /logs [On][Off] - Activa o desactiva el sistema de logs.
  • /setup - Configura el bot en el servidor.
  • /edit - Edita la configuración de un sistema.
